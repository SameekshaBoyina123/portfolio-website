## üìö Brief Explanation of HTML Concepts Learned

### 1) What is HTML?
HTML (HyperText Markup Language) is used to create the structure of web pages. It uses tags and elements to display content in the browser.

### 2) HTML Page Structure
This project follows proper HTML5 structure:
- `<!DOCTYPE html>` defines HTML5 document
- `<html>` root tag
- `<head>` contains meta information and title
- `<body>` contains visible webpage content

### 3) Tags Used in This Project
- Headings: `<h1>`, `<h2>`, `<h3>`
- Paragraphs: `<p>`
- Links: `<a>`
- Images: `<img>`
- Lists: `<ul>`, `<ol>`, `<li>`
- Sections: `<section>`
- Forms: `<form>`, `<input>`, `<textarea>`, `<button>`

### 4) Attributes Used
- `id` for section navigation
- `href` for internal & external links
- `src` for image path
- `alt` for image description
- `required` for form validation
- `placeholder` for input hints

### 5) Semantic HTML Used
This project uses semantic tags:
- `<header>` - Website title and navigation
- `<nav>` - Navigation links
- `<main>` - Main content container
- `<section>` - Different sections of page
- `<footer>` - Copyright information
